Textual rule: CBPR_Message_Pagination_TextualRule

When message pagination is used, the message must contain only one report / statement / notification. Please, see separate XML-sample for rule validation. There is no recommended solution proposed for handling an Entry that is bigger than a file size constraints between customer, bank and the channel being used. In this case it is recommended that it be handled on communications level (e.g. multiple compressed files)
