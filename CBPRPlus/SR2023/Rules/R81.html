May be used to indicate type of Notification.
Where this is used, all transactions within this message are of the same type.
Codes are fro example:
/LBOX/ - Lock box
/BULK/ - Bulk reporting (batch transaction with underlying transactions)
/RTRN/ - Return report
/CRED/ - Notification with Credit entries ONLY. /CRED/ will be provided by the Bank to the Corporate in case of Instant Payment receivable
