Textual rule: CBPR_DEBT_Rule_1_TextualRule

If Instructed amount and Interbank Settlement amount are expressed in the same currency:

If Charge Bearer/DEBT is used then charge information is only mandatory in case of prepaid charges (that is if interbank settlement amount is higher than instructed amount) and in that case zero amount is not allowed. Otherwise Charge information is optional (both Agent and currency always need to be provided).

This rule only applies when Interbank Settlement Amount and Instructed amount are expressed in the same currency
